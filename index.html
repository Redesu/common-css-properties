CSS Table Reference

CSS Properties

Property Name Description
color: Color Specify the text color of an HTML element.
font-size: Font size Change the size of the font for any text based content
background-color: background-color Set the background color of an HTML element.
border: Border Set a border for your HTML element
background-image: Background Image Add a background image to an HTML element.
border-radius: Border-Radius Add rounded corners of a HTML elements
padding: Padding An element's padding is the space between its content and its border.
margin: Margin Sets the margins for an element
border: Border Set the style of the borders for different elements.
